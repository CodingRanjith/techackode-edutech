## 1. AI poster / image prompts (optional)

Headline area reserved at top, bottom strip for logo and URL. Colors: deep blue and teal accents, white background, professional, not cluttered, no illegible small text.

**Prompt C — social square**

> Square 1080x1080, bold sans-serif headline block, icon row for laptop chat calendar, soft gradient background, tech education brand, modern startup aesthetic, space for ₹ pricing badge.
